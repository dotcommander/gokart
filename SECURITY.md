# Security policy

## Supported versions

GoKart is pre-1.0. Security fixes are provided on the latest released minor version. Upgrade to the latest release before reporting a problem that is already fixed there.

## Reporting a vulnerability

Please use [GitHub private vulnerability reporting](https://github.com/dotcommander/gokart/security/advisories/new) so details remain private while the issue is assessed. Do not open a public issue for a suspected vulnerability.

Include the affected package or command, version, reproduction steps, impact, and any suggested mitigation. You should receive an initial response within seven days.

Do not include real credentials, private data, or destructive proof-of-concept payloads.
